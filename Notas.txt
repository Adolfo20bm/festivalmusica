- Para instalar SASS y sus dependencias
npm install SASS
- Para instalar SASS y sus dependencias en entorno de desarrollo
npm install SASS --save-dev
- Puedes borrar la carpeta node_modules y volver a instalarlo con
npm i  ya que el archivo package.json es el que contiene todo y no lo debes borrar
- Instalamos gulp
npm i -D gulp
- Lllamamos el gulpfile
npx gulp primerTarea
- Si quieres llamar desde el package.json
npm run primerTarea
- Instalar gulp-sass
npm i --save-dev gulp-sass
- Corremos gulp 
npx gulp css
npm run css    esto si es que lo agregaste al package.json
- Instalar plumber
npm i --save-dev gulp-plumber
- Dependencia gulp-webp
npm install --save-dev gulp-webp
- Dependencia para aligerar imagenes 
npm i --save-dev gulp-imagemin@7.1.0
npm i --save-dev gulp-cache
